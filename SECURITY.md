# Security Policy

## Supported versions

Cowboy currently supports security fixes for the latest released version on
`main`.

| Version | Supported |
| ------- | --------- |
| Latest release | Yes |
| Older releases | No |

## Reporting a vulnerability

Do not open public GitHub issues for security reports.

Preferred path:

1. Use GitHub Private Vulnerability Reporting or a GitHub Security Advisory if
   private reporting is enabled for the repository.
2. If private reporting is not enabled, contact the maintainers through a
   private email address or contact channel listed on the GitHub organization,
   maintainer profile, or package registry metadata.

Please include:

- A clear description of the issue
- Affected versions or commit range
- Reproduction steps or a proof of concept
- Expected impact
- Any suggested mitigation

The maintainers will acknowledge receipt, assess impact, and coordinate a fix
and disclosure timeline.
